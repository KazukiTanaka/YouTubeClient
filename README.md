# YouTubeClient

This is a sample using ``GoogleAPIClientForREST`` and ``youtube-ios-player-helper``.

## Installation
1. Run ``pod install``
2. Add an APIKey ``YouTubeClient/Models/VideoPlaylistFactory.swift`` 

```
service.apiKey = "YOUR_API_KEY"
```
